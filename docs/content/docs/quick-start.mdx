Get OpenTabs running and make your first tool call in under 5 minutes.

<QuickStartFlow />

## 1. Install the CLI

```bash
npm install -g @opentabs-dev/cli
```

## 2. Start the server

```bash
opentabs start
```

On first run, you'll see the MCP endpoint URL, log file path, Chrome extension setup instructions, and ready-to-copy configuration snippets for popular MCP clients (Claude Code, Cursor, Windsurf, OpenCode) — with your auth secret pre-filled.

On first run, this also creates the `~/.opentabs/` directory, generates an auth secret, and installs the Chrome extension files.

## 3. Load the Chrome extension

1. Open `chrome://extensions/` in Chrome
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked** and select `~/.opentabs/extension`

The extension connects to the server automatically. You should see the OpenTabs icon in your toolbar.

## 4. Configure your MCP client

When `opentabs start` ran in step 2, it printed ready-to-copy MCP client config blocks — with your auth secret already filled in. Copy the block for your client and merge it into your MCP client config file:

For Claude Code, the easiest way is the CLI:

```bash
claude mcp add --transport http opentabs http://127.0.0.1:9515/mcp \
  --header "Authorization: Bearer YOUR_SECRET_HERE"
```

Or merge the printed JSON block into `~/.claude.json` under `"mcpServers"` manually.

| Client | Config file |
| --- | --- |
| Claude Code | `~/.claude.json` (merge into `"mcpServers"`) |
| Cursor | `.cursor/mcp.json` in your project root |
| Windsurf | `~/.codeium/windsurf/mcp_config.json` |
| OpenCode | `~/.config/opencode/opencode.json` |

To see the config blocks again at any time:

```bash
opentabs start --show-config
```

Or to view just the current config:

```bash
opentabs config show --show-secret
```

## 5. Install a plugin

Find available plugins:

```bash
opentabs plugin search
```

Install it:

```bash
opentabs plugin install slack
```

## 6. Make a tool call

Ask your AI agent to use the plugin's tools. The agent will see all registered tools from your installed plugins, plus 40 built-in browser tools for tab management, screenshots, DOM interaction, and more.

Verify everything is connected:

```bash
opentabs status
```

```bash
opentabs doctor
```

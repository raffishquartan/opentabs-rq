# Components

This page demonstrates Fumadocs MDX components styled with the RetroUI neobrutalist design system.

## Callout Variants

<Callout type="info" title="Info Callout">
  This is an **info** callout. Use it to highlight useful information for the reader.
</Callout>

<Callout type="warn" title="Warning Callout">
  This is a **warning** callout. Use it to alert the reader to something important.
</Callout>

<Callout type="error" title="Error Callout">
  This is an **error** callout. Use it to highlight critical issues or breaking changes.
</Callout>

<Callout type="success" title="Success Callout">
  This is a **success** callout. Use it to confirm that something worked correctly.
</Callout>

<Callout type="idea" title="Idea Callout">
  This is an **idea** callout — useful for tips, suggestions, and creative notes.
</Callout>

  This callout has no type (defaults to **info**) and no title — just a body message.
</Callout>
